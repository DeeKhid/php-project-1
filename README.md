# Добро пожаловать в Brain Game!
Игра созданная разогреть ваш мозг!
### Установка
```
make install
```
### Всего 5 игр:
1. Четное число или нет
```
make brain-even
```
2. Посчитать пример
```
make brain-calk
```
3. Определить НОД (наибольший общий делитель)
```
make brain-gcd
```
4. Найти член арифметической прогрессии
```
make brain-progression
```
5. Просто число или нет
```
make brain-prime
```
